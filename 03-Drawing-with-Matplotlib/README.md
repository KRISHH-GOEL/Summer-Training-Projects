# 🏠 Hut Drawing using Matplotlib

A simple Python graphics project that draws a colorful hut using the **Matplotlib** library. This project demonstrates that Matplotlib can be used not only for plotting graphs but also for creating simple drawings using coordinates.

## 📖 Project Overview

The hut is created by plotting multiple line segments and adding a circular window using Matplotlib. It is a beginner-friendly project that helps in understanding:

- Coordinate-based drawing
- Line plotting
- Different colors
- Circular markers
- Figure customization

## ✨ Features

- 🏠 Hut structure
- 🔺 Triangular roof
- 🚪 Door
- 🪟 Circular window
- 🎨 Different colors for different parts
