# 🌸 Flower Species Prediction using Machine Learning

A simple Machine Learning web application that predicts the species of an Iris flower using a **Logistic Regression** model. The application is built with **Python**, **Scikit-Learn**, and **Streamlit**.

---

## 📌 Project Overview

This project classifies an Iris flower into one of the following species based on its measurements:

- 🌷 Setosa
- 🌼 Versicolor
- 🌺 Virginica

The model was trained using the famous **Iris Dataset** and deployed as an interactive web application using Streamlit.

## 🛠 Technologies Used

---

## 🌸 Input Features

The model uses four input features:

- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Flower-Species-Prediction.git
```

### 2. Navigate to the Project Folder

```bash
cd Flower-Species-Prediction
```

### 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit App

```bash
streamlit run flower_app.py
```
