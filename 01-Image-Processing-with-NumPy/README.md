# Image Processing with NumPy

A beginner-friendly Computer Vision project demonstrating how digital images can be represented as NumPy arrays and manipulated using Python.

This project explores fundamental image processing operations such as RGB channel visualization, pixel-level manipulation, and custom color transformations using NumPy, Pillow, and Matplotlib.

---

## Project Overview

Digital images are nothing more than multidimensional arrays of pixel values. This project demonstrates how image processing works by treating an image as a NumPy array and performing various transformations directly on pixel values.

The project is designed to strengthen the understanding of:

- Image representation in Python
- RGB color model
- Pixel manipulation
- NumPy array operations
- Basic Computer Vision concepts

---

## Features

- Load an image using Pillow
- Convert image into NumPy array
- Display image dimensions
- Inspect pixel values
- Manipulate RGB colors
- Visualize Red channel
- Visualize Green channel
- Visualize Blue channel
- Display processed outputs using Matplotlib

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/01-Image-Processing-with-NumPy.git
```

Move into the project directory

```bash
cd 01-Image-Processing-with-NumPy
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python image_processing.py
```
