# 💰 Salary Prediction using Decision Tree Regression

A Machine Learning web application that predicts an employee's salary based on their **Years of Experience** using a **Decision Tree Regressor**. The application is built with **Python**, **Scikit-Learn**, and **Streamlit** for an interactive user interface. :contentReference[oaicite:0]{index=0}

---

## 📌 Project Overview

This project demonstrates a complete Machine Learning workflow:

- Data preprocessing
- Model training
- Model evaluation
- Saving the trained model
- Deploying the model using Streamlit

The user simply enters their years of experience, and the application predicts the expected salary.

---

## 🚀 Features

- 📊 Predict salary based on years of experience
- 🌳 Decision Tree Regression model
- 💻 Interactive Streamlit interface
- ⚡ Instant salary prediction
- 💾 Pre-trained model loading using Joblib

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Salary-Prediction.git
```

Move into the project directory

```bash
cd Salary-Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit app

```bash
streamlit run salary_app.py
```

---

## 💻 How to Use

1. Launch the Streamlit application.
2. Enter the **Years of Experience**.
3. Click **Predict Salary**.
4. View the predicted salary instantly.
